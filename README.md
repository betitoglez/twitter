# twitter
Integration with Twitter API

Ejecutar el script por consola:
================================

    1. Instalar la base de datos 
    		
    		mysql -uAdmin -p < script.sql
    		
	2. Lanzar el comando
	
			php run.php "Nombre del Hashtag"
	
	
Ver los resultados por HTML:
================================

    1. Otorgar permisos de escritura a la carpeata temporal /tmp
    2. Lanzar /html/index.php 
    
    
Sistemas de Caché:
=================================

	1. Caché a nivel de plantilla vía Twig 
	2. Caché a nivel de objeto usando Memcache, los datos se regeneran cada 15 minutos
